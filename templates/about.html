Database Schema

Collections

1. Devices

Stores only device IDs.

    {
        "_id": "device_123",
        "created_at": "2025-02-02T12:00:00Z"
    }

2. Experiment

Name of the experiment, options with their probabilities.

    {
        "_id": "65b8c1f2a9e8f7e2a8d5b9f1",
        "name": "button_color",
        "options": {
            "#FF0000": 0.75,
            "#00FF00": 0.1,
            "#0000FF": 0.15
        },
        "created_at": "2025-02-02T12:00:00Z"
    },
    {
        "_id": "65b8c1f3b2a8f7e2a8d5b9f2",
        "name": "price",
        "options": {
            "10": 0.75,
            "20": 0.1,
            "50": 0.05,
            "5": 0.1
        },
        "created_at": "2025-02-03T14:30:00Z"
    }

3. Experiment Assignments

Maps devices to assigned experiments. Example:

    {
        "device_id": "fake_token_123",
        "experiments": {
            "button_color": "#00FF00",
            "price": 20
        },
        "assigned_at": "2025-02-02T12:05:00Z"
    }
